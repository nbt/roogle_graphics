RoogleGraphics::Shape

A Shape element defines a closed polygon to be rendered.

Public Class Methods

new(origin_x, origin_y, points, opts = {}) click to toggle source

Define a new shape, to be rendered at origin_x, origin_y. points is an array of x,y pairs, where each xy pair is an offset from origin_x, origin_y, as in:

[[x0,y0],[x1,y1],[x2,y2], ...[x0,y0]].

Recognized options:

outline_color

the color of the shape's outline, as a six-character hex string. Default is nil (no outline).

fill_color

the fill color for the shape as a six-character hex string. Default is nil (no fill).

# File roogle_graphics.rb, line 44
def initialize(origin_x, origin_y, points, opts = {})
  super(origin_x, origin_y)
  @points = points || []
  @outline_color = opts[:outline_color]
  @fill_color = opts[:fill_color]
  # make sure polygon is closed
  @points << @points.first unless (@points.first == @points.last)
end
