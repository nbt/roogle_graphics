new(opts = {}) click to toggle source

Create a new, empty Plot object to which you can add Text and Shape elements. Recognized options are:

:width

width of the plot in pixels (default: 300)

:height

height of the plot in pixels (default: 300)

:fill_color1

primary fill color as 6-char hex string. (default: 'ffffff', i.e. white)

:fill_color2

gradient fill color. (default: :fill_color1)

:angle

gradient fill angle in degrees (default: 0, i.e. left to right)

# File roogle_graphics.rb, line 100
def initialize(opts = {})
  @width = opts[:width] || 300
  @height = opts[:height] || 300
  @elements = []
  @fill_color1 = opts[:fill_color1] || 'ffffff'
  @fill_color2 = opts[:fill_color2]
  @angle = opts[:angle] || 0
end

Push a new graphic element (Text or Shape) on to the display list. In general, elements are drawn in the order they are added, so the last element to be added is always drawn on top.

generate_uri() click to toggle source

Emit a URI, suitable as an HTTP GET command, that will return the graphic, generated courtesy of Google's graphing web service.

# File roogle_graphics.rb, line 119
def generate_uri
  [GOOGLE_CHART_SERVICE + '?cht=lxy',
   generate_dimensions,
   inhibit_axes,
   generate_line_colors,
   generate_lines,
   generate_markers,
   generate_background
  ].join('&')
end
