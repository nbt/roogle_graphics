Description

roogle_graphics is a Ruby module that renders simple text and shapes into .png graphics by calling the Google Chart web service.

Note: roogle_graphics largely subverts the Google Chart API's intented function of generating charts from quantitative data. If you want to generate quantitative plots, please refer to:

code.google.com/apis/chart/docs/chart_params.html
